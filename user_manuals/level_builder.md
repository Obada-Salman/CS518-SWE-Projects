# Level Builder User Manual  

## Overview
The level builder is interative map desiner that allows deveolper and player to create maps.

## How to use
Load into level builder from the main menu level builder button.

The level builder works by left clicking on spaces on the map to place tiles, and right click to remove them.  

The aviable tiles are shown on the right hand margin. These tiles can be selected by left clciking on them.  

To scroll across the map use the 'd' key to scroll to the right  and use the 'a' key to scroll to the left  

To change the level use the arrow keys, the right arrow key '->' will increase the level and the left arrow key '<-' will decrease the level.  

There are three buttons on the button margin.  

The level type button displays the current level type you have selcted (community or story), clicking the button will change the level type and reload the map.  

The save button will save the current map, but the map can only be saved as long as the player and goal has been placed.  

The exit button returns you to the main menu, any unsaved progress will be lost.
